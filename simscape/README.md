# Matlab-Simulink Model

## Description 

This is a repository for the OwnTech matlab-simulink models.

It contains all the tools necessary to its usage.

The model simulates a "Buck" power conversion.

## Getting started

Make sure you have : 

- Matlab
- Simulink
- Simscape
- Power System Blocks

## How to use

- Open the configuration script "script_owntech_model.m"
- Configure the parameters and values as you need
- Run the script

## Observe existing results

- Open the script "Plot_compare_Core_Simscape.m" in the folder "results"
- Configure the parameters as you need
- Run the script

## Contributing

- You can fork this git project
- ALWAYS work on a different branch from the main
